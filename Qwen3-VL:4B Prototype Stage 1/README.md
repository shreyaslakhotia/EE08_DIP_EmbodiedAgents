# Study Buddy AI Agent
A multimodal prototype using Qwen 3-VL for vision, hearing, and reasoning.

## Setup
1. **Ollama:** Install [Ollama](https://ollama.com) and run `ollama pull qwen3-vl:4b`.
2. **Environment:** Run the `setup` script for your OS (`.sh` for Mac, `.bat` for Windows).

## Usage
- **Chat:** Type prompts in the bottom entry box and press Enter.
- **Voice:** Speak directly to the agent; it listens and transcribes automatically. You can also say the following keywords under "Sight"
- **Sight:** Say keywords like "Look", "See", "show", "analyze", "watch", "describe my face", "what do you see" to trigger the camera analysis.
